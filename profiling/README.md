# profiling

프로파일링 스택입니다. 현재는 디렉토리만 생성되어 있으며, 서비스 구성은 예정 중입니다.

## 예정된 아키텍처

```
애플리케이션
    │ pprof / pyroscope SDK
    ▼
┌──────────────┐
│   Pyroscope  │  지속적 프로파일링 데이터 수집 및 저장
└──────┬───────┘
       │
       ▼
┌──────────────┐
│   Grafana    │  Flame graph 시각화
└──────────────┘
```

지속적 프로파일링(Continuous Profiling)을 통해 CPU, 메모리, goroutine 등의 성능 병목을 시간 흐름에 따라 분석할 수 있습니다.
